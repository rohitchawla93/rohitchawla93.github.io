# Rohit Chawla — Dark Research Portfolio

Static HTML/CSS website for GitHub Pages.

## How to edit quickly

- Homepage text: `index.html`
- Design/colors/layout: `assets/css/style.css`
- Research page: `research.html`
- Project gallery: `projects.html`
- Publications: `publications.html`
- Contact details: `contact.html`

## Where to put future media

- Videos/animations: `assets/videos/`
- Images/CAD renders: `assets/img/`
- PDFs/CV/lab sheets: `assets/docs/`

## Suggested workflow

Ask ChatGPT for edits in plain English, for example:

- "Add my robot arm video to the Projects page."
- "Make the homepage less text-heavy."
- "Add my new PRE paper to Publications."
- "Add a pseudo-arclength continuation project page."

Then replace the updated files in your GitHub Pages repository.
